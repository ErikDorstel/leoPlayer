# leoPlayer
ESP32 based children's music box
#### Features
#### GPIO MAX98357A I2S
* GPIO 16 - DIN
* GPIO 17 - BCLK
* GPIO 25 - WSCLK
#### GPIO WS2812B LED ring
* GPIO 26 - DIN
#### GPIO SD Card
* GPIO 19 - MISO
* GPIO 23 - MOSI
* GPIO 18 - CLK
* GPIO 5 - CS
